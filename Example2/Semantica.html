Semantica

las etiquetas semanticas son auellas que transmiten un significado a lo que contienen, las etiquetas semanticas solo tiene la funcionalidad de contener otras etiquetas e indentificarlas


esta etiqueta se usa para contener todos los elementos/etiquetas de la cabeza de pagina

esta etiqueta se usa para contener todos los elementos/etiquetas que se utilizen para la navegacion

esta etiqueta se usa para contener todos los elementos/etiquetas principales de la pagina

esta etiqueta se usa para contener todos los elementos/etiquetas que se definen por separado de la pagina y que tiene sentido por si mismos

esta etiqueta se usa para contener todos los elementos/etiquetas que son secciones de la pagina

esta etiqueta se usa para contener todos los elementos/etiquetas que son separados del contenido de la pagina (publicidad)

esta etiqueta se usa para contener todos los elementos/etiquetas del pie de pagina

esta etiqueta se utiliza para contener elementos/etiquetas dentro de otros elementos/etiquetas identificandolos con un id o una clase
